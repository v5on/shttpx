## ğŸ›  How to Use

### 1. File Sharing (Default)
1. Open shttpx.
2. Select a folder containing your files.
3. Click **START SERVER**.
4. Share the displayed URL (e.g., `http://192.168.0.5:8080`) with others connected to the same Wi-Fi.

### 2. Website Hosting
1. Create a folder with your `index.html`, `style.css`, etc.
2. Open shttpx Settings âš™ï¸.
3. Enable **"Host Website"**.
4. Select your folder and Start Server.
5. Your website is now live on your local network!
